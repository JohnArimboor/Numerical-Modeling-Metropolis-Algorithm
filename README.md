# Numerical-Modeling-Metropolis-Algorithm

The aim of this project is to use simulation to calculate the thermodynamic properties (like average energy and heat capacity) of a physical system. Instead of solving extremely complex calculus equations by hand, we use a smart guessing game the Metropolis Monte Carlo algorithm to simulate how a particle behaves at a specific temperature.

In this specific project I have aimed to implement a numerical simulation using the Metropolis-Hastings Monte Carlo algorithm to calculate the parameters of a classical limit system .A particle is attached to a microscopic spring (a 1D harmonic oscillator) sitting in a room at 300 K. It is constantly jittering around due to thermal energy.

Problem to solve:-Find the particle's average potential energy and heat capacity. For simple systems, we can do this on paper. But for complex gases or molecular networks, the math becomes impossible.
