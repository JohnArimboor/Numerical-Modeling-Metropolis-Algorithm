# Numerical-Modeling-Metropolis-Algorithm

The aim of this project is to use python simulation to calculate the thermodynamic properties (like average energy and heat capacity) of a physical system. For simple systems, we can do this on paper. But for complex gases or molecular networks, the math becomes impossible.

In this specific project I have aimed to implement a numerical simulation using the Metropolis-Hastings Monte Carlo algorithm to calculate the parameters of a classical limit system.By systematically sampling the configuration space, the simulation computes the average potential energy, average square energy, and heat capacity of the system, verifying the results against theoretical frameworks like the Equipartition Theorem.
